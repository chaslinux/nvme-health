# nvme-health
Check the health of an NVMe drive

<img width="1920" height="1080" alt="Image" src="https://github.com/user-attachments/assets/73e0aad3-1629-48da-8eb2-63afcbece8fb" />

By default this script checks the first drive at /dev/nvme0n1. It's a simple script that will evolve over time. It's worth noting that WARNings like the one above about ERRORs do not necessrily mean a drive is bad, you may need to look deeper using other tools.
